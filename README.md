# Investment Agent

Personal AI-powered portfolio monitor for a Fidelity account (~$108K across two accounts). 

---

## Quick Start

```bash
# 1. Backend
source venv/bin/activate
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# 2. Dashboard (separate terminal)
cd dashboard && npm run dev

# 3. Open
open http://localhost:5173
```

**First run:** Drop your Fidelity CSV exports (`Portfolio_Positions_*.csv`) into the project root. The backend loads them automatically on startup, or click **Reload CSV** in the Portfolio tab.

| `FINNHUB_API_KEY` | Free tier | https://finnhub.io/register | Real-time prices, company news, earnings estimates, analyst ratings |

| `EDGAR_API_KEY` | Free | https://app.edgar.tools/ | SEC filings (10-K, 10-Q, 8-K, insider trades) via the edgar.tools MCP server — used by Alpha Agent's Research Agent |
| `TELEGRAM_BOT_TOKEN` | Free | Message `@BotFather` on Telegram → `/newbot` | Morning briefs, stop-loss alerts, Telegram commands |
| `TELEGRAM_CHAT_ID` | Free | Message `@userinfobot` on Telegram | Targets all alerts to your personal chat |

### Investor Profile Variables

These are not API keys — they shape how the agent sizes positions and manages risk.

| Variable | Default | Description |
|---|---|---|
| `PORTFOLIO_SIZE` | `100000` | Total portfolio value in USD — used for position sizing calculations |
| `RISK_TOLERANCE` | `balanced` | `conservative`, `balanced`, or `aggressive` |
| `MIN_HOLD_DAYS` | `30` | Minimum hold period before the agent suggests selling |
| `MAX_SINGLE_POSITION_PCT` | `8` | Maximum % of portfolio in any one stock |
| `MAX_SECTOR_CONCENTRATION_PCT` | `35` | Maximum % of portfolio in any one sector |

### Minimum Keys to Get Running

| Key | Without it |
|---|---|
| `ANTHROPIC_API_KEY` | Nothing works — all AI features disabled |
| `FINNHUB_API_KEY` | No market data sync, no real-time prices |
| `FRED_API_KEY` | No VIX data, regime gate defaults to NORMAL |
| `FINANCIAL_MODELLING_API_KEY` | No technicals, fundamentals, or earnings calendar |

All other keys degrade gracefully — the feature using that API logs a warning and skips.

## Web App Guide

### Header (always visible)

| Element | What it shows |
|---|---|
| **Portfolio value** | Total across both accounts, refreshes every 60s |
| **VIX badge** | Market stress level — green <15, yellow 15–20, orange 20–30, red 30+ |
| **API cost badge** | Today's Anthropic spend vs $0.15 daily budget |
| **? button** | Replays the guided walkthrough tour |

---

### PORTFOLIO tab

Your holdings overview across both Fidelity accounts.

- **Today's Pick card** (top) — Claude analyses the top momentum screener result not in your portfolio. Refreshes once daily. Click **Details** to see risks and the recommended action.
- **Holdings table** — all positions with current value, all-time gain/loss %, and risk status:
  - `STOP LOSS` — position down 8%+ from cost basis (immediate alert)
  - `SOFT REVIEW` — position down 5%+ (review suggested)
  - `TARGET 1` — position up 10%+ (consider taking profit)
  - `TARGET 2` — position up 18%+ (strong take-profit signal)
- **Sector Concentration pie** — visual check against the 35% sector cap
- **Reload CSV button** — re-parse your Fidelity CSV files after downloading fresh exports

---

### SIGNALS tab

Buy / sell / watch signals generated by Claude Haiku.

- **Score bar** — 10 blocks; green = strong buy (7–10), yellow = watch (4–6), red = sell (1–3)
- **Generate signal** — type any ticker and press Enter or click the button; signal is stored in the DB
- **Expand row** — click any signal row to read Claude's full reasoning, confidence level, and risks
- Signals from your scanned portfolio appear here automatically (runs daily)

---

### EVENTS tab

Live feed from four data sources:

| Source | What it covers |
|---|---|
| **gdelt** | Geopolitical events that could affect markets |
| **edgar** | SEC 8-K, 10-Q, 10-K filings for your holdings |
| **rss** | News from Yahoo Finance, MarketWatch, CNBC, Reuters |
| **finnhub** | Company-specific news |

- **Filter buttons** — click a source name to filter the feed
- Events are colour-coded by tone (positive = green, negative = red, neutral = grey)
- Impact score 1–10 shown on each event

---

### ANALYSIS tab

Three tools, all powered by Claude.

#### Morning Brief
Generates a full portfolio summary — market regime, top watch items, actionable points.

- **Generate Brief** — generates and displays in the browser
- **Generate & Send to Telegram** — same as above, plus pushes the formatted brief to your `@abk_invest_bot`

#### Deep Dive
Type any ticker and click **Deep Dive** for a Sonnet-powered analysis covering:
- Verdict (strong buy / buy / hold / sell / strong sell) with score
- Technical summary (RSI, MACD, Bollinger Bands, SMA)
- Fundamentals (P/E, EPS growth, beta)
- Key price levels (support, resistance, stop, target)
- Full reasoning and risks

#### YTD Coaching
Click **Get Coaching Report** for a Sonnet review of your year-to-date performance — grade, wins, areas to improve, and actionable advice.

---

### TRADE LOG tab

History of all logged trades.

**To log a trade** use your Telegram bot:
```
/bought NVDA 10 875.00   ← buy 10 shares at $875
/sold   AAPL 5  195.00   ← sell 5 shares at $195
```

Trades appear in this tab immediately. The agent uses this history for Kelly Criterion position sizing.

---

### CONFIG tab

Adjust all agent settings live — changes take effect on the next analysis run.

| Group | Settings |
|---|---|
| **Risk Thresholds** | Stop loss % (−8), soft review % (−5), target 1 % (+10), target 2 % (+18) |
| **Position Limits** | Max single position % (8), max sector concentration % (35), min hold days (30) |
| **Budget** | Daily API budget ($0.15), scan abort threshold ($0.12) |
| **Notifications** | Max daily Telegram messages (8), Telegram enabled on/off |

Click **Save [Group]** after changing values in each section.

> Scheduler intervals (market data every 15 min, VIX daily, etc.) are fixed in code — restart the backend to change them.

## Updating Your Portfolio

1. Log into Fidelity → Accounts & Trade → Portfolio → Download CSV
2. Save the file as `Portfolio_Positions_<date>.csv` in the project root
3. Click **Reload CSV** in the Portfolio tab (or it auto-reloads every 30 minutes)

The agent supports multiple CSV files simultaneously (both your Individual TOD and Roth IRA accounts).

## Cost Tracking

- Target: **under $0.15/day** in Anthropic API costs
- Haiku (routine scans): ~$0.001 per signal
- Sonnet (deep dives, coaching): ~$0.01–0.03 per call
- Scans auto-abort if today's spend reaches $0.12
- View live cost in the header badge or at `GET /analysis/daily-cost`
